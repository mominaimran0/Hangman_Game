# Hangman_Game
A simple text-based Hangman game built with Python.

## Description
The player has to guess a randomly selected word one letter at a time. The game allows a maximum of **6 incorrect guesses** before the player loses.

## Features
- Randomly selects a word from a predefined list
- Displays guessed letters and hidden letters
- Tracks incorrect guesses
- Maximum of 6 wrong attempts
- Console-based gameplay

## Concepts Used
- Random module
- Lists
- Strings
- While loop
- If-else statements
- User input

## How to Run
1. Clone this repository.
2. Open the project in Python.
3. Run the Python file.

```bash
python hangman.py
```
